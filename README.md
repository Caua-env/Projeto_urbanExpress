🚀 UrbanExpress - Landing Page

Landing page desenvolvida com foco em apresentação de um serviço de entregas rápidas, com design moderno, responsivo e organizado.

📌 Sobre o projeto

Este projeto foi criado com o objetivo de praticar desenvolvimento front-end, utilizando HTML e CSS para construir uma interface limpa, funcional e responsiva.

A página apresenta seções comuns em produtos reais, como:

Hero (apresentação principal)
Benefícios do serviço
Como funciona
Call to Action (CTA)
Prova social
Rodapé
🛠️ Tecnologias utilizadas
HTML5
CSS3
Responsividade (Media Queries)
🎯 Objetivo

Desenvolver habilidades em:

Estruturação de páginas web
Organização de layout
Responsividade
Boas práticas de desenvolvimento
📱 Responsividade
